Hack The Holidays was an event held on 613 Global Way, Linthicum Heights, MD we competed in. CC did relatively okay, placing 5th out of 8 in a competiton that involved larger company teams. Below we are going to break down a few of the problems we liked, but it's important to note a few problems relied on tools like DirBuster, which brute force. When networking is a concern, CC does not brute force in any fashion, so we didn't last long in 1st place. Please note some of this is off of memory.

CC scored a total of approximately 875. Problems ranged from 50-500 points, with them mostly coming in two stages: One problem to get read access on a box, one to get root on that box. The total was as such:

  • Subdomain Hijack - Value: 500 (only one team was allowed to solve this)
  • Web User - Value: 100
  • Web Admin - Value: 150
  • Git User 2 - Approximate Value: ?150?
Problems we did not get included:
  • Freebie User - Value: 50
  • Freebie Admin - Value: 50
  • Encryptor Admin - Value: 100
  • Git User - Value: 125
  • Storage User - Value: 150
  • Storage Admin - Value: 150
  • Sarah User - Value: 200
  • Sarah Admin - Value: 200
  • Git Admin - Value: 250
  • Encryptor User - Value: 275
